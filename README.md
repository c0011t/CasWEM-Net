# CasWEM-Net

Accurate retinal vessel segmentation remains challenging due to low contrast, blurred boundaries, and complex morphology. This paper proposes CasWEM-Net, a cascaded Wavelet-Mamba network with edge guidance. A three-stage cascaded collaborative strategy integrates multi-scale features via global-to-local refinement. The WaveMamba block couples spatial selective scanning with multi-level discrete wavelet transform to capture long-range context while preserving high-frequency vessel details. A bidirectional attention skip connection (BASC) with edge-guided channel attention (EGCA) bridges the semantic gap by enabling reverse information flow from decoder to encoder, enhancing fine-vessel connectivity. Experiments on DRIVE, CHASE_DB1, and STARE show that CasWEM-Net achieves strong performance in sensitivity (Se) and F1-score, demonstrating robustness for thin vessel segmentation and pathological interference.
